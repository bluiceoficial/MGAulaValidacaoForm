Este repositório faz parte de uma aula prática realizada ao vivo na Twitch e no YouTube.

## 🎯 Objetivo

Este projeto foi desenvolvido durante uma live com o objetivo de ensinar como trabalhar com **validação de formulários sem redirecionamento**, utilizando:

* PHP (backend)
* JavaScript puro (frontend)

A proposta é mostrar como exibir mensagens de erro diretamente na página, melhorando a experiência do usuário.
